# vue-demo

## Project setup
```
npm install
```

### 启动前端 有热加载（编辑完前端代码会自动编译刷新页面）
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```

### Customize configuration
See [Configuration Reference](https://cli.vuejs.org/config/).

### 启动后端）
```
node index
```
